### asciinema
#### 1.asciinema install
- PyPi
```bash
sudo pip3 install asciinema
```

- On Linux
```bash
sudo apt-get install asciinema
```

#### 2.asciinema use
| 命令                     | 解释         |
|--------------------------|--------------|
| asciinema rec            | 开始录制     |
| `Ctrl`+`d`               | 停止录制     |
| asciinema rec demo.cast  | 保存到本地   |
| asciinema play demo.cast | 播放录制视频 |
